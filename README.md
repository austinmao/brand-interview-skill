# Brand Interview Skill

A guided conversational brand-discovery interview that produces two complete documents for your business: `BRAND.md` (strategy, positioning, audience, narrative, voice) and `DESIGN.md` (color, typography, spacing, and component tokens).

Runs fully standalone in any Claude session — no setup required beyond loading the skill. If you have the Glance brand connector installed and authorized, it automatically upgrades to a connected mode that saves your answers as you go and can persist the final documents to your brand corpus.

Built on named brand-strategy frameworks (StoryBrand, Jobs to Be Done, positioning-statement format, brand archetype, emotional-arc mapping) — see `SKILL.md` → **Frameworks Applied** for exactly how.

## Get started

See **[tenant-setup-instructions.md](tenant-setup-instructions.md)** for the 3 ways to use this:

- **Option A — claude.ai Project** (recommended, no technical setup)
- **Option B — Custom GPT** (ChatGPT)
- **Option C — GitHub** (Claude Code, or claude.ai's Skills feature) — clone or download this repo directly
